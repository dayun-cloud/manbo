# 🎵 曼波配音生成器


## 📝 使用说明

- **输入文字**：在文本框中输入要转换的文字内容
- **生成语音**：点击生成按钮，将调用API进行语音合成
- **播放音频**：生成成功后可直接在页面播放音频
- **复制链接**：点击复制按钮可获取音频文件链接
- **下载音频**：在音频播放组件右边更多选项里点击下载

## ⚙️ API 说明

项目使用的语音合成API：
- **接口地址**：`https://api.milorapart.top/apis/mbAIsc`
- **请求方式**：GET
- **参数**：`text` (URL编码的文字内容)
- **返回格式**：JSON
